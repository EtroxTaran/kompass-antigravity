# File Organization Rules

## Root Directory
- **Allowed**: `docs/`, `.agent/`, `.github/`, config files (`package.json`, `tsconfig.json`, etc.), `README.md`, `CHANGELOG.md`, `LICENSE`.
- **Forbidden**: Documentation files in root (move to `docs/`), Backup files (`*.bak`, `*_old`), Temp files.

## Documentation
- **Location**: All documentation must be in `docs/` and its subdirectories (`api/`, `architecture/`, `guides/`, `processes/`, `specifications/`).
